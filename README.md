## ✨ Features

- 📷 **Barcode Scanning:** Instantly scan packaged food barcodes using your device's camera.
- 📊 **Health Insights:** Get immediate access to the official Nutri-Score and NOVA classification for comprehensive health ratings.
- 🗃️ **Extensive Database:** Powered by the Open Food Facts API with an intelligent fallback strategy for regional Indian brands.
- 🎨 **Premium UI/UX:** Enjoy a sleek, system-aware dark mode, haptic feedback, and smooth SVG micro-animations.
- ⌨️ **Manual Entry:** A seamless manual entry fallback mechanism for products when the camera isn't accessible.
- ⚡ **Skeleton Loading:** Shimmering skeleton states ensure the app feels incredibly fast and responsive while data is fetched.

## 🛠️ Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6+)
- **API:** Open Food Facts API
- **Deployment:** GitHub Pages

## 🚀 Getting Started Locally

Want to run the project on your own machine? It's simple.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Jit-Halder/NutriScan
   ```
2. **Navigate into the directory:**
   ```bash
   cd food-barcode-scanner
   ```
3. **Open `index.html`** directly in your browser or run a local live server.
